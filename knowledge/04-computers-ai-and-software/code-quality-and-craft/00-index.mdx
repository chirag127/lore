# ✨ Code Quality & Craft

## 📚 Overview

Writing code that works is easy. Writing code that's clean, maintainable, and a joy to work with is hard. This shelf covers the practices and disciplines that separate professional code from amateur code: automated testing, code review, refactoring, and readable code.

These books are essential for anyone who writes code — from junior developers learning the craft to senior engineers looking to raise their standards.

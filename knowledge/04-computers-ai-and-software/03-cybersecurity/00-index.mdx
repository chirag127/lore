# 🔒 Cybersecurity

## 📚 Overview

In a world where everything is connected, security is not optional. This shelf covers the major topics in cybersecurity: cryptographic primitives, network security, public key infrastructure, binary analysis, and penetration testing.

These books provide both the theoretical foundations and the practical skills needed to understand and defend against digital threats.

## 📖 Topics Covered

- Cryptographic primitives: encryption, hashing, digital signatures
- Network security and threat modeling
- Public key infrastructure and certificate authorities
- Practical binary analysis and reverse engineering
- Systems security and penetration testing

## ⭐ Essential Books

- **Practical Binary Analysis** — Dennis Andriesse: How to analyze binaries for security vulnerabilities.
- **Cryptographic Primitives** — The building blocks of secure communication: encryption, hashing, and digital signatures.
- **Network Security and Threat Modeling** — How to identify and mitigate security threats in networked systems.
- **Public Key Infrastructure and Certificate Authorities** — How the internet's trust model works.
- **Systems Security and Penetration Testing** — How to test systems for vulnerabilities before attackers do.

## 🚀 Why This Category Matters

Cybersecurity is one of the most critical challenges of the digital age. Understanding how attacks work and how to defend against them is essential for anyone who builds or manages digital systems.

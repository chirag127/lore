# 💻 Computer Science

## 📚 Overview

Computer science is the study of computation: what can be computed, how efficiently, and with what tools. This shelf covers the foundational topics — from how CPUs and memory work to the theory of programming languages, from operating systems to functional and object-oriented programming.

These books provide the deep understanding that separates engineers who can use tools from those who understand how and why those tools work.

## 🎯 Scope

| Included | Excluded |
|----------|----------|
| CPU architecture and memory hierarchy | Algorithms (→ Data & Algorithms) |

## 🔗 Category Boundaries

- **Data & Algorithms** covers algorithms and data structures.
- **Software Engineering** covers the practice of building software.
- **Artificial Intelligence** covers machine learning and AI.

This category focuses on the **foundations of computing** — the hardware, systems, and theory that underlie all software.

## 📖 Topics Covered

- CPU architecture and instruction sets
- Memory hierarchy and cache design
- Operating systems: process scheduling, memory management
- File systems and I/O
- Functional programming principles
- Object-oriented programming principles
- Systems programming languages (C, C++, Rust)
- Theory of programming languages
- Virtualization and containerization

## ⭐ Essential Books

- **CPU Architecture** — How processors work, from instruction pipelines to branch prediction.
- **Memory Hierarchy and Cache Design** — Why memory access patterns matter and how caches speed things up.
- **Process Scheduling and Memory Management** — How operating systems manage resources.
- **Functional Programming** — The paradigm that's reshaping how we think about code.
- **Object-Oriented Programming Principles** — The foundations of OOP: encapsulation, inheritance, polymorphism.

## 🗺 Related Categories

- **Data, Algorithms & Databases** — Algorithms and data structures
- **Software Engineering** — Building software systems
- **Networking & Distributed Systems** — Networked computing
- **Theory of Computation** — Theoretical foundations

## 🚀 Why This Category Matters

Understanding computer science fundamentals makes you a better engineer regardless of what you build. These books provide the deep knowledge that helps you debug hard problems, optimize performance, and make better architectural decisions.

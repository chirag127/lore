# 🌐 Networking & Distributed Systems

## 📚 Overview

The internet is the most complex system ever built. This shelf covers how it works: from the TCP/IP protocol stack to routing and switching, from high-performance browser networking to wireless protocols and mobile networks.

These books provide the deep understanding of networking that every systems engineer and architect needs.

## 🔗 Category Boundaries

- **System Design & Architecture** covers the design of distributed systems.
- **Cybersecurity** covers network security.
- **Software Engineering** covers network programming.

This category focuses on the **networking protocols and infrastructure** that enable distributed computing.

## 📖 Topics Covered

- TCP/IP protocol stack: from physical layer to application layer
- Routing, switching, and network infrastructure
- High-performance browser networking
- Wireless networks and mobile protocols
- Network troubleshooting and analysis

## ⭐ Essential Books

- **Computer Networking: A Top-Down Approach** — James Kurose: The most widely used networking textbook, with a top-down approach that starts with applications.
- **TCP/IP Illustrated, Volume 1** — Kevin Fall: A detailed, visual guide to the TCP/IP protocols.
- **High-Performance Browser Networking** — Ilya Grigorik: How to build fast, efficient web applications by understanding the network.
- **Network Warrior** — Gary Donahue: A practical guide to network infrastructure for working engineers.
- **Wireless Networks and Mobile Protocols** — How Wi-Fi, Bluetooth, and cellular networks work.

## 🚀 Why This Category Matters

Every application you build communicates over a network. Understanding how networks work — from TCP/IP to HTTP, from routing to congestion control — is essential for building fast, reliable, and scalable systems.
